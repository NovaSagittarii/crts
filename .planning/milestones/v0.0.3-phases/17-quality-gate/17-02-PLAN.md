3. **Test: "Grid.toPacked() ArrayBuffer survives Socket.IO binary attachment path (QUAL-01)"**
   - Use the `test` fixture to get `connectedRoom` and `startLockstepMatch`
   - Start match: `const match = await startLockstepMatch(connectedRoom)`
   - Get the `room:joined` payload from `match.hostJoined` (or `match.guestJoined`) -- the `state` field contains the `grid` ArrayBuffer
   - Assert `match.hostJoined.state.grid instanceof ArrayBuffer` -- proves Socket.IO transmitted it as binary, not mangled to a JSON-serialized form
   - Unpack the received grid: `const receivedGrid = Grid.fromPacked(match.hostJoined.state.grid, 52, 52)`
   - Get the server-side grid state for comparison: tick the match clock 0 times (the initial state is the comparison), then get server state by requesting state via the existing test infrastructure, OR simply validate that the unpacked grid has the expected dimensions and cell count
   - **Key validation:** The unpacked grid must have valid cell states (all 0 or 1) and the correct dimensions. Compare using `Uint8Array`:
     ```typescript
     const receivedBytes = new Uint8Array(match.hostJoined.state.grid);
     // Verify the ArrayBuffer is not empty and has the correct byte length
     // For a 52x52 grid, packed size = ceil(52*52 / 8) = 338 bytes
     expect(receivedBytes.byteLength).toBe(Math.ceil((52 * 52) / 8));
     ```
   - **Round-trip validation:** Unpack the received ArrayBuffer into a Grid, then re-pack it, then compare bytes:
     ```typescript
     const unpacked = Grid.fromPacked(match.hostJoined.state.grid, 52, 52);
     const repacked = unpacked.toPacked();
     expect(new Uint8Array(repacked)).toEqual(
       new Uint8Array(match.hostJoined.state.grid),
     );
     ```
   - This proves: Socket.IO binary attachment -> client receives ArrayBuffer -> Grid.fromPacked -> Grid.toPacked round-trips perfectly
