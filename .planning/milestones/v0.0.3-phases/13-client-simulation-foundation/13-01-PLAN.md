<objective>
Add `RtsRoom.fromPayload()` to `packages/rts-engine` -- a static factory that reconstructs a fully tickable `RtsRoom` from a `RoomStatePayload` plus template list. This is the shared, runtime-agnostic core that both the browser client (Phase 13) and future reconnect replay (Phase 16) depend on.

Purpose: Without this factory the client cannot initialize a local simulation from the server-provided state snapshot. Every downstream lockstep feature depends on payload-based room reconstruction producing bit-identical determinism hashes.

Output: `RtsRoom.fromPayload()` method in rts.ts; unit tests in rts.test.ts proving hash equivalence.
</objective>

    Implementation steps (following the createShadowRoom pattern from apps/server/src/server.ts but adapted for wire payload):

    1. Build `templateMap` from the provided `templates` array:
       ```typescript
       const templateMap = new Map<string, StructureTemplate>();
       for (const template of templates) {
         templateMap.set(template.id, template);
       }
       ```

    2. Compute `spawnOrientationSeed` via `RtsEngine.hashSpawnSeed(payload.roomId, payload.width, payload.height)`.

    3. Create `RoomRuntime` via `createRoomRuntime({ id: payload.roomId, name: payload.roomName, width: payload.width, height: payload.height, templateMap, spawnOrientationSeed })`.

    4. Reconstruct the grid from the packed ArrayBuffer:
       ```typescript
       const grid = Grid.fromPacked(payload.grid, payload.width, payload.height);
       ```

    5. Create the bare `RoomState` object (same pattern as `createRoomState`):
       ```typescript
       const room = {
         generation: payload.generation,
         tick: payload.tick,
         grid,
         templates,
         teams: new Map<number, TeamState>(),
         players: new Map<string, RoomPlayerState>(),
       } as unknown as RoomState;

       defineRoomRuntimeProperties(room, runtime);
       attachRoomRuntime(room, runtime);
       ```

    6. Sort teams by id and reconstruct each team. For each `TeamPayload` (sorted by `teamPayload.id` ascending):
       - Sort structures by key before inserting into the `structures` Map.
       - Reconstruct each `Structure` from `StructurePayload`: look up template via `templateMap.get(sp.templateId)`, call `template.instantiate({ key: sp.key, x: sp.x, y: sp.y, transform: sp.transform, active: sp.active, isCore: sp.isCore })`, then set `structure.hp = sp.hp` to preserve damage.
       - Reconstruct `pendingBuildEvents` from `PendingBuildPayload[]`. For each `pb`:
         ```typescript
         const buildEvent: BuildEvent = {
           id: pb.eventId,
           teamId: teamPayload.id,
           playerId: pb.playerId,
           templateId: pb.templateId,
           x: pb.x,
           y: pb.y,
           transform: pb.transform,
           executeTick: pb.executeTick,
           reservedCost: templateMap.get(pb.templateId)?.activationCost ?? 0,
         };
         ```
       - Reconstruct `pendingDestroyEvents` from `PendingDestroyPayload[]`. For each `pd`:
         ```typescript
         const destroyEvent: DestroyEvent = {
           id: pd.eventId,
           teamId: teamPayload.id,
           playerId: pd.playerId,
           structureKey: pd.structureKey,
           executeTick: pd.executeTick,
         };
         ```
       - Reconstruct `TeamState`:
         ```typescript
         const team: TeamState = {
           id: teamPayload.id,
           name: teamPayload.name,
           playerIds: new Set<string>(teamPayload.playerIds),
           resources: teamPayload.resources,
           income: teamPayload.income,
           incomeBreakdown: { ...teamPayload.incomeBreakdown },
           lastIncomeTick: payload.tick,  // correct for post-tick snapshots
           territoryRadius: DEFAULT_TEAM_TERRITORY_RADIUS,
           baseTopLeft: { ...teamPayload.baseTopLeft },
           defeated: teamPayload.defeated,
           structures,
           pendingBuildEvents,
           pendingDestroyEvents,
           buildStats: { queued: 0, applied: 0, rejected: 0 },
         };
         ```
       - Add each player to `room.players`:
         ```typescript
         for (const playerId of teamPayload.playerIds) {
           room.players.set(playerId, { id: playerId, name: playerId, teamId: teamPayload.id });
         }
         ```
       - Insert team into `room.teams` Map (already sorted).
       - Call `reserveRoomTeamId(room, teamPayload.id)` to keep `nextTeamId` correct.
