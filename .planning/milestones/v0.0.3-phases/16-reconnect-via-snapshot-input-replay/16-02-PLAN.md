    6. Guest reconnects: `const guest2 = connectClient({ sessionId: 'recon-guest' })`
    7. Capture reconnect payload: `const rejoined = await waitForEvent<RoomJoinedPayload>(guest2, 'room:joined', 5000)`
    8. Assert:
       - `expect(rejoined.inputLog).toBeDefined()`
       - If `buildQueued.executeTick > rejoined.state.tick`, then the build event should be in the inputLog (it was queued after the last snapshot tick and its executeTick is in the future):
         ```typescript
         if (buildQueued.executeTick > rejoined.state.tick) {
           expect(rejoined.inputLog!.length).toBeGreaterThanOrEqual(1);
           const buildEntry = rejoined.inputLog!.find(e => e.kind === 'build');
           expect(buildEntry).toBeDefined();
         }
         ```
       - If `buildQueued.executeTick <= rejoined.state.tick`, the build is already reflected in the snapshot. The inputLog from tick+1 correctly excludes it: `expect(rejoined.inputLog!.filter(e => e.kind === 'build' && (e.payload as BuildQueuedPayload).eventId === buildQueued.eventId).length).toBe(0)`
