    **Create `packages/bot-harness/live-bot-strategy.ts`:**

    This module is runtime-agnostic (no Socket.IO). It wraps TF.js model inference.

    ```typescript
    import * as tf from '@tensorflow/tfjs';
    import type { RoomStatePayload } from '#rts-engine';
    import type { ObservationResult } from './observation-encoder.js';
    import { PayloadObservationEncoder } from './payload-observation-encoder.js';
    import { ActionDecoder } from './action-decoder.js';
    ```

    Export `class LiveBotStrategy`:
    - Constructor: `(model: tf.LayersModel | null, width: number, height: number)`.
    - Private fields: `encoder: PayloadObservationEncoder`, `decoder: ActionDecoder`, `model`, `lastActionIndex: number | null`.
    - `infer(payload: RoomStatePayload, teamId: number, maxTicks: number): number`:
      - If model is null, return random action index (0 to width*height*5, uniformly).
      - Encode observation via `this.encoder.encode(payload, teamId, maxTicks)`.
      - Create tensors: transpose planes from [C,H,W] to [H,W,C] via `tf.tensor3d(obs.planes, [channels, height, width]).transpose([1, 2, 0]).expandDims(0)`. Create scalars tensor `tf.tensor2d(obs.scalars, [1, obs.shape.scalarCount])`.
      - Run `model.predict([planesTensor, scalarsTensor])` to get `[logits, value]`.
      - Apply simplified action mask: check if team has resources >= 5 (minimum build cost). If not, force action 0 (no-op). The server validates all builds anyway.
      - Sample action: `tf.multinomial(tf.softmax(logits), 1).dataSync()[0]`.
      - Dispose ALL tensors (planes, scalars, logits, value, softmax, multinomial).
      - Store as `lastActionIndex`, return it.
    - `getLastAction(): number | null`: returns `lastActionIndex` for cached fallback.
    - `warmUp(): void`: Run inference with zero-filled tensors of correct shape. Dispose outputs. Logs "Model warmed up" to stderr.
    - `decode(actionIndex: number): BuildQueuePayload | null`: delegates to `this.decoder.decode(actionIndex)`.

    ```typescript
    #!/usr/bin/env tsx
    import { parseArgs } from 'node:util';
    import { io } from 'socket.io-client';
    import type { Socket } from 'socket.io-client';
    import type {
      ClientToServerEvents,

      RoomJoinedPayload,

      RoomStatePayload,
      MatchFinishedPayload,
      RoomMembershipPayload,
    } from '#rts-engine';
    ```

    CLI arguments via `parseArgs`:
    - `--server` (string, default `'http://localhost:3000'`)
    - `--room` (string, required -- room ID or room code)
    - `--slot` (string, default `'team-2'`)
    - `--model` (string, optional -- explicit model path)
    - `--runs-dir` (string, default `'runs'`)
    - `--fallback` (string, default `'noop'`)
    - `--budget-ms` (string, default `'80'`)
    - `--single-match` (boolean, default false)
    - `--verbose` (boolean, default false)
    - `--name` (string, default `'Bot (PPO)'`)
    - `--help`, `-h` (boolean)

    Main async flow:
    1. Parse args, show help if requested.
    2. Load model: `const model = await loadBotModel({ modelPath, runsDir }).catch(() => null)`. If null, log warning "No trained model found, using random actions" to stderr.
    3. Create `TickBudgetTracker` with `{ budgetMs, fallback }`.
    4. Connect socket: `io(serverUrl, { auth: { sessionId: botSessionId ?? undefined }, transports: ['websocket'], reconnection: true, reconnectionAttempts: Infinity, reconnectionDelay: 1000, reconnectionDelayMax: 10000 })`. If no explicit sessionId, let server assign one.
    5. Track state: `roomStatus`, `teamId: number | null` (initialized to `null`), `gridWidth`, `gridHeight`, `inferring` (boolean to skip stale ticks).

    Socket event handlers:
    - `'connect'`: Log "Connected to server". Emit `'room:join'` with roomId or roomCode. Emit `'player:set-name'` with `{ name: botName }`.
    - `'room:joined'` (payload: RoomJoinedPayload): Store `teamId = payload.teamId` (may be `null` if slot not yet claimed), `gridWidth = payload.state.width`, `gridHeight = payload.state.height`. Create `LiveBotStrategy` with model and dimensions. If slot specified, emit `'room:claim-slot'` with `{ slotId }`.
    - `'room:slot-claimed'` (payload: RoomSlotClaimedPayload): **Update `teamId = payload.teamId`** (this resolves the null from room:joined). Then emit `'room:set-ready'` with `{ ready: true }`. Log "Slot claimed, teamId: ${teamId}".
    - `'room:membership'` (payload: RoomMembershipPayload): Update `roomStatus = payload.status`.
    - `'room:match-started'`: Set `roomStatus = 'active'`. Call `strategy.warmUp()` if model loaded. Log "Match started, team ${teamId}".
    - `'state'` (payload: RoomStatePayload): **Guard: if `teamId === null`, log warning "Received state but teamId is null, skipping" to stderr and return.** Only act if `roomStatus === 'active'` and `!inferring` and `teamId !== null`. Set `inferring = true`. Run observe-infer-act:
      ```
      budgetTracker.startTick();
      const actionIndex = strategy.infer(payload, teamId, maxTicks);
      const metrics = budgetTracker.endTick();
      if (verbose) console.error(budgetTracker.formatMetricsLog(metrics, payload.tick));
      if (budgetTracker.shouldAct(metrics)) {
        const buildPayload = strategy.decode(actionIndex);
        if (buildPayload) socket.emit('build:queue', buildPayload);
      } else if (fallback === 'cached') {
        const lastAction = strategy.getLastAction();
        if (lastAction !== null && lastAction !== 0) {
          const buildPayload = strategy.decode(lastAction);
          if (buildPayload) socket.emit('build:queue', buildPayload);
        }
      }
      inferring = false;
      ```
    - `'build:queue-rejected'`: If verbose, log rejection reason to stderr.
    - `'room:match-finished'` (payload: MatchFinishedPayload): Log match result. Print tick budget stats. If `singleMatch`, disconnect and exit. Else set `roomStatus = 'finished'` and wait.
    - `'disconnect'`: Log "Disconnected, reconnecting...".
    - `'connect_error'`: Log error to stderr.

    Exit handling: `process.on('SIGINT', () => { socket.disconnect(); process.exit(0); })`.

    The `maxTicks` value should default to `DEFAULT_MAX_TICKS` from `#bot-harness` (2000).

    **Critical implementation details:**
    - Use `tf.tidy()` to wrap the inference pipeline inside `LiveBotStrategy.infer()` to prevent tensor leaks (Pitfall 1).
    - Transpose [C,H,W] to [H,W,C] before model input (Pitfall 2).
    - Track `inferring` flag to skip state events that arrive during inference (Pitfall 5).
    - **teamId null safety:** `teamId` is `number | null` throughout. Initialize from `room:joined` (may be null pre-slot-claim). Update to resolved value in `room:slot-claimed`. Guard the `state` handler: `if (teamId === null) return;` -- do not pass null to `strategy.infer()` which expects `number`.
    - Do NOT import Socket.IO in `packages/bot-harness/`. All Socket.IO code is in `bin/play-bot.ts` only.
