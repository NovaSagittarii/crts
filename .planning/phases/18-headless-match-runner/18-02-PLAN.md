    **1. Create `packages/bot-harness/match-runner.ts`** implementing:

    **`createBotView(room: RtsRoom, teamId: number, tick: number): BotView`**

    - Find the team with matching `teamId` from `payload.teams`
    - Map it to `TeamStateView` (own team only, per D-02 -- opponent team data is excluded)
    - Get template summaries from `room.state.templates.map(t => t.toSummary())`

    - For each action in `actions`:
      - If `action.type === 'build'` and `action.build` exists: call `room.queueBuildEvent(playerId, action.build)`
      - If `action.type === 'destroy'` and `action.destroy` exists: call `room.queueDestroyEvent(playerId, action.destroy)`

    **`createTickRecord(tick: number, result: RoomTickResult, room: RtsRoom, botActions: [BotAction[], BotAction[]], teamIds: [number, number], hashCheckpointInterval: number): TickRecord`**
    - Collect `actions` from `result.buildOutcomes` and `result.destroyOutcomes` -- map each to `TickActionRecord` with `teamId`, `actionType`, `templateId`, `x`, `y`, `transform` (the transform from the outcome), `result` (applied ? 'applied' : reason string), `structureKey`
    - Collect `economy` from `room.createStatePayload().teams` -- map each to `TickEconomyRecord` with `teamId`, `resources`, `income`
    - Set `buildOutcomes: result.buildOutcomes.length`, `destroyOutcomes: result.destroyOutcomes.length`
    - If `tick % hashCheckpointInterval === 0` (per D-07): call `room.createDeterminismCheckpoint()` and set `hash: checkpoint.hashHex`
    - Return `TickRecord` object

    **`runMatch(config: MatchConfig, botA: BotStrategy, botB: BotStrategy, callbacks?: MatchCallbacks): MatchResult`**
    - Derive room ID via `seedToRoomId(config.seed)` (per D-09, D-16)
    - Create room: `RtsRoom.create({ id: roomId, name: \`headless-${roomId}\`, width: config.gridWidth, height: config.gridHeight })`
    - Add two players: `room.addPlayer('bot-a', botA.name)` -> `teamA`, `room.addPlayer('bot-b', botB.name)` -> `teamB`
    - Tick loop (per D-04, D-09, D-10, D-14):
      ```
      for tick = 0; tick < config.maxTicks; tick++:
        viewA = createBotView(room, teamA.id, tick)
        viewB = createBotView(room, teamB.id, tick)
        actionsA = botA.decideTick(viewA, teamA.id)
        actionsB = botB.decideTick(viewB, teamB.id)
        applyBotActions(room, 'bot-a', actionsA)
        applyBotActions(room, 'bot-b', actionsB)
        result = room.tick()
        tickRecord = createTickRecord(tick, result, room, [actionsA, actionsB], [teamA.id, teamB.id], config.hashCheckpointInterval)
        callbacks?.onTickComplete?.(tick, tickRecord)
        if result.outcome !== null: break
      ```
    - After loop: construct `MatchResult` with `seed`, `config`, `outcome` (from last tick result or null), `totalTicks`, `bots: [botA.name, botB.name]`, `isDraw: outcome === null`
    - Call `callbacks?.onMatchComplete?.(matchResult)`
    - Return `matchResult`
    - Do NOT hold references to RoomState or Grid in the returned result -- only serializable data (per Pitfall 5)
