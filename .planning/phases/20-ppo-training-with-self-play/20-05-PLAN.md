3. Create `bin/train.ts` CLI entry point following the same pattern as `bin/run-matches.ts`:
   ```typescript
   #!/usr/bin/env tsx
   ```

   a. Import `parseTrainingArgs`, `generateRunId` from `#bot-harness`.
   b. Import `TrainingCoordinator` from `#bot-harness`.
   c. Parse args via `parseTrainingArgs()`. If `--help`, print usage and exit.
   d. Generate run ID via `generateRunId()` (or use `--resume` run ID if provided per D-11).
   e. Print startup banner: run ID, key config values (episodes, workers, learning rate, grid size).
   f. Create coordinator: `new TrainingCoordinator(config)`.
   g. Await `coordinator.init()`.
   h. Set up graceful shutdown: handle SIGINT/SIGTERM to call `coordinator.cleanup()` and exit.
   i. Await `coordinator.run()`.
   j. Await `coordinator.cleanup()`.
   k. Print completion summary: total episodes, final win rate, run directory path.

   Help text should list all flags with defaults, matching bin/run-matches.ts style:
   ```
   Usage: tsx bin/train.ts [options]

   PPO Training with Self-Play

   Options:
     --episodes <n>         Total training episodes (default: 1000)
     --lr <n>               Learning rate (default: 0.0003)
     --clip-epsilon <n>     PPO clip epsilon (default: 0.2)
     --gamma <n>            Discount factor (default: 0.99)
     --gae-lambda <n>       GAE lambda (default: 0.95)
     --ppo-epochs <n>       PPO update epochs per batch (default: 4)
     --mini-batch-size <n>  Mini-batch size (default: 64)
     --entropy-coeff <n>    Entropy coefficient (default: 0.01)
     --value-loss-coeff <n> Value loss coefficient (default: 0.5)
     --max-grad-norm <n>    Max gradient norm (default: 0.5)
     --target-kl <n>        Target KL for early stopping (default: 0.015)
     --workers <n>          Worker threads (default: auto-detect)
     --batch-episodes <n>   Episodes per collection round (default: workers*4)
     --output-dir <dir>     Output directory (default: runs)
     --resume <run-id>      Resume from existing run
     --conv-filters <list>  Conv filter sizes, comma-separated (default: 32,64,64)
     --mlp-units <list>     MLP hidden units, comma-separated (default: 256,128)
     --latest-ratio <n>     Self-play latest opponent ratio (default: 0.5)
     --historical-ratio <n> Self-play historical opponent ratio (default: 0.3)
     --random-ratio <n>     Self-play random opponent ratio (default: 0.2)
     --checkpoint-interval <n> Episodes between checkpoints (default: 50)
     --max-pool-size <n>    Max opponent pool size (default: 30)
     --grid-width <n>       Grid width (default: 52)
     --grid-height <n>      Grid height (default: 52)
     --max-ticks <n>        Max ticks per episode (default: 2000)
     -h, --help             Show this help
   ```
