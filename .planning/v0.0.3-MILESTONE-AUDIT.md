## Tech Debt

### Phase 13: Client Simulation Foundation
- **reservedCost divergence** in `ClientSimulation.applyQueuedBuild()` — computes `template.activationCost` instead of reading `payload.reservedCost`. Causes hash mismatch on every live build event, triggering SYNC-02 resync. System self-heals but produces unnecessary resync traffic. Fix: change line 134 to `reservedCost: payload.reservedCost ?? template?.activationCost ?? 0`.

### Phase 16: Reconnect via Snapshot + Input Replay
- **pendingSimInit not cleared** in `room:match-finished` handler. Nil impact due to status guard but minor state-cleanliness gap.
